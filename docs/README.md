## はじめに

このアプリは、Dart, Flutter の勉強のために個人開発したアプリです。

そのため、**稚拙なアーキテクト設計や実装を行っている場合**があります。

## このアプリについて

このアプリでは、以下のことができます。

- ユーザー情報(ユーザー名、カラーテーマ)の設定
- Places API を使った場所の検索、サジェスト、地図の表示
- 検索した場所の登録、削除
- 広告の表示
